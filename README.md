# 退休財務規劃 Dashboard 📈

這是一套純前端（HTML5 + JavaScript）構建的單頁式應用程式（SPA），專為高階財務規劃設計。系統捨棄了傳統 Excel 繁瑣的 VBA 與靜態圖表，透過瀏覽器即時進行數千次矩陣運算，並整合 `Chart.js` 提供專業且互動性高的視覺化報表。本系統不僅支援\*\*三種確定性情境（最佳/一般/最差）\*\*的長期資產推演，更內建了 \*\*10,000 次蒙地卡羅模擬（Monte Carlo Simulation）\*\*來計算專屬您的退休成功率。

## ✨ 核心特色

* **⚡ 零伺服器依賴**：完全在瀏覽器端運行，不需架設後端、不需資料庫，保護財務隱私資料絕不外洩。
* **🎯 退休收入空窗期（Retirement Gap）模擬**：可分別設定「退休年齡」與「年金給付年齡」，精準模擬中斷收入卻尚未領取年金的現金流低谷。
* **🎲 蒙地卡羅機率引擎**：導入 `Box-Muller` 常態分配轉換演算法，一鍵執行 10,000 次隨機市場波動模擬，計算「目標年齡」內的財務安全機率。
* **📊 專業級視覺化分析**：內建 5 張高互動圖表（資產走勢、年度現金流、通膨購買力侵蝕、資產成長來源、花費結構）。
* **🖨️ 友善列印設計**：自動隱藏側邊欄與按鈕，點擊列印即可匯出完美的 A4 格式 PDF 報表，適合理財顧問與客戶展示。

## 🛠️ 技術棧

* **核心語言**：HTML5, CSS3, 原生 JavaScript (Vanilla JS)
* **視覺化套件**：[Chart.js](https://www.chartjs.org/) (透過 CDN 引入)

## 🚀 快速開始

本專案無需任何安裝流程（NPM / Webpack 等皆不需要）。

1. 將本專案下載至您的電腦。
2. 找到 `retirement\_dashboard.html` 檔案。
3. 點擊右鍵，選擇\*\*「透過瀏覽器開啟」\*\*（建議使用 Google Chrome, Microsoft Edge 或 Safari）。
4. 即可開始在左側輸入您的財務參數。

## 📝 關鍵參數說明

在左側的參數設定區中，有幾個進階參數的填寫方式需要特別注意：

### 1\. 退休年齡 vs 退休年金給付年齡

* **預計退休年齡**：停止工作、停止每年本金投入的年齡。
* **退休年金給付年齡**：開始領取勞保、勞退或其他常態性退休收入的年齡。
* *範例：設定 55 歲退休，65 歲領年金。系統會在 55\~64 歲間自動停止收入，完全從總資產中扣除生活開銷。*

### 2\. 單次大額支出 (格式規範)

可用於規劃換車、房屋修繕、子女結婚或出國環遊世界等單筆大筆開銷。此數字為**現值**，系統會自動幫您加上通膨計算。

* **格式**：`年齡,金額` (使用半形逗號分隔，多筆之間使用半形分號 `;` 隔開)
* **範例**：`65,100; 75,50` (代表在 65 歲時有一筆 100 萬的支出，75 歲時有一筆 50 萬的支出)

### 3\. 蒙地卡羅目標年齡

計算「蒙地卡羅成功率」的終點線。若您只要求資產能支撐到 85 歲不歸零即可，請將此參數設為 85，系統在跑 10,000 次亂數迴圈時，只要資產在 85 歲前沒有破產，即判定該次模擬成功。

## 🧠 演算法架構簡述

本系統雙軌並行兩種財務模型：

1. **確定性模型 (Deterministic Model)**

   * 即時運算（On-Change）。
   * 基於使用者輸入的通膨率、以及三個固定的報酬率（最佳 7%、一般 5%、最差 3%），計算出現金流與破產年齡。
2. **隨機性模型 (Stochastic Model / Monte Carlo)**

   * 手動觸發（Click-to-Run），避免 UI 卡頓。
   * 以「一般情境報酬率」為平均期望值 (μ)，搭配「投資組合波動度」為標準差 (σ)。
   * 運用 `Math.random()` 搭配 `Box-Muller Transform` 產出當年度的隨機報酬率，逐年推進資產變化，共執行一萬次人生財務軌跡。

## ⚠️ 免責聲明

本系統提供之計算結果僅供財務規劃參考，不構成任何投資建議。實際金融市場具備高度不確定性與黑天鵝風險（如極端通膨、市場崩盤等），使用者應自行承擔投資決策之風險。建議搭配專業財務顧問進行全面性評估。
